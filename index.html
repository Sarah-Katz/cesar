Chiffrement de césar

Ce code réalise une version informatique du chiffrement de César, une technique ancienne pour rendre les messages secrets. Il utilise des boutons pour "Encoder" et "Décoder", des champs de texte en entrée et sortie, et un champ pour le mot de passe.

Lorsque tu appuies sur "Encoder", les lettres du message avancent dans l'alphabet selon une clé numérique générée à partir du mot de passe. Quand tu appuies sur "Décoder", les lettres reculent pour révéler le message original.

La clé numérique est calculée en additionnant les valeurs numériques des lettres du mot de passe. Pour chaque lettre du texte, sa valeur numérique est modifiée selon la clé pour obtenir le message chiffré ou déchiffré.
